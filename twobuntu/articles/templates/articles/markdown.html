Markdown Help

Articles that are composed and written for the 2buntu blog are written in a text-formatting language known as "Markdown". The language is both easy to read and easy to learn. What follows is a relatively brief overview of Markdown syntax and some examples that demonstrate its usage.

Simple Text Formatting

**bold** | *italic* | _also italic_

The Markdown above produces the results below when rendered:

bold | italic | also italic

You can create headings with the following syntax:

# Top Level Heading
## Second Level Heading

An alternative syntax is also supported:

Heading
=======

Links to URLs

Links to other pages can be created using the syntax shown below:

[link text](http://2buntu.com)

This produces the following results:

link text

Bare URLs will also be automatically converted to links.

Images

Embedding images in posts is very simple:

![hover text](http://example.com/image.jpg)

The "hover text" is displayed when the user hovers the cursor over the image.
